# School Bell System
## About
### Stack
1. RPi3 B
2. 3.5 mm audio system
3. Python + Flask + Sqlite3
### Installation
Clone this repository to your Raspberry PI</br>
```
git clone https://github.com/ret7020/SchoolBellProject
```
Start the installer(WITH ROOT PRIVILAGES)</br>
```
sudo ./installer.sh
```
Run project</br>
```
./run.sh
```
